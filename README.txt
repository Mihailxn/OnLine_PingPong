Описание приложения:
Игра "On-line ping-pong" - классивеский вариант игры из 90х - "Pong",
но с тремя режимами игры:
1) Режим онлайн игры с другим игроком.
2) Режим офлайн многопользовательской игры.
3) Режим офдайн игры с ботом.

Описание режимов:
1) Режим онлайн игры требует наличия второго игрока, подкдлючённого к серверу.
Если второго игрока нет, то пользовтнль ожидает его подключения или выбирает другой режим.
Сервер поддерживает множество пар игроков.
Для управления игроками используются стрелки "вверх" и "вниз".
2) Режим офлайн многопользовательской игры представлят собой классическую игры на двух пользователей с одной клавиатуры.
Для управления первым игроком используются стрелки "вверх" и "вниз", а вторым клавиши "W" и "S".
Так же присутствует возможность паузы при нажатии кнопки "P".
3) Режим офдайн игры с ботом - игра против электронного бота, который отбивает мячик с вероятностью 0,7.
Для управления игроком используются стрелки "вверх" и "вниз".
Так же присутствует возможность паузы при нажатии кнопки "P".

Для выхода из режимов используется кнопка "ESC".
